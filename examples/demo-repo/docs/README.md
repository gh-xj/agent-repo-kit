# Docs

Tracked docs root for `demo-repo`. This repo uses the intent-first taxonomy
declared in `.convention-engineering.json`.

- `requests/` — RFIs, asks, scope, and acceptance criteria.
- `planning/` — design decisions and architecture choices.
- `plans/` — executable implementation plans.
- `implementation/` — rollout notes, verification notes, and implementation reports.
- `taxonomy/` — stable vocabularies, schemas, and mappings.

External-source-backed material belongs in `.wiki/`, not `docs/`.
